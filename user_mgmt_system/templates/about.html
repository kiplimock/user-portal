What You Can Do

Create...

a new user by simply entering the name, email and password and clicking "register".

Read...

the contents of your database by first logging in and going to the users page. You will see a list of all the users registered with you.

any user using the edit button associated with the user you wish to edit. Once you're satisfied with the editing, submit the new information.

Delete...

any user by clicking the delete button adjacent to the user you wish to delete and confirming your action.
